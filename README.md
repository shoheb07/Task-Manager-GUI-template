# Task-Manager-GUI-template
simple Task Manager GUI template in Python using Tkinter. It allows you to add, delete, and mark tasks as completed.

📝 Task Manager (Python Tkinter)

📌 Overview

This is a simple Task Manager application built using Python and Tkinter. It provides a graphical user interface (GUI) to manage daily tasks efficiently. Users can add tasks, delete them, and mark tasks as completed.


🚀 Features

- Add new tasks
- Delete selected tasks
- Mark tasks as completed
- Scrollable task list
- Simple and user-friendly interface


🛠️ Technologies Used

- Python 3
- Tkinter (built-in Python GUI library)


📂 Project Structure

task_manager.py   # Main application file
README.md         # Project documentation

▶️ How to Run the Application

1. Install Python
   
   - Download from: https://www.python.org/
   - Ensure Python is added to PATH

2. Download or Clone the Project
   
   git clone <repository-link>
cd task-manager

3. Run the Application
   
   python task_manager.py


📷 Application Workflow

1. Enter a task in the input field
2. Click Add Task to add it to the list
3. Select a task and:
   - Click Delete Task to remove it
   - Click Mark Completed to mark it as done

⚠️ Notes

- Tasks are not saved permanently (they will be lost when the app is closed)
- Ensure you select a task before deleting or marking it
